# dotfiles

## Requirements

- [Zsh](https://www.zsh.org)
- [Oh My Zsh](https://github.com/robbyrussell/oh-my-zsh) as zsh configuration manager
- Git

## Installation

1. Clone:

   ```bash
   git clone https://github.com/spodin/dotfiles.git ~/.dotfiles
   cd ~/.dotfiles
   ```

2. Stuff Zsh with custom theme and plugins using 'Oh My Zsh':

   `source ./configure_zsh.sh`
   
3. Install dotfiles:

   `./install.sh`
